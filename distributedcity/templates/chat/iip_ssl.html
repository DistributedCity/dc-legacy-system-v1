IIP SSL Interface
Distributed City's Stunnel interface to IIP will allow users of SSL IRC clients to acces IIP without installing a local IIP proxy.

In the near future, Distributed City will offer an IIP server available via SSL. This will allow users who are unable to set up a local IIP proxy to access IIP via SSL enabled IRC clients.

The availability of this service will be announced in the DC forums. More information will be available here at that time.
