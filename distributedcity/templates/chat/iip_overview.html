Chat Overview
Distributed City uses the Invisible Internet Relay Chat system (part of the Invisible Internet Project - IIP ) as its platform for providing secure pseduo-anonymous realtime chat services to DC users.

Until an inline DC IIP Web Interface is implemented, you will need to install the IIP Proxy client application (ispoxy) on your machine as well as a standard Internet Relay Chat (IRC) client to access IIP chat - see: Installing IIP & IRC.

Once connected you should join one of the DC IIP Channels.

Understanding the Invisible IRC Project (IIP)

The IIP isproxy application runs on your machine which emulates an IRC server (called an IRC daemon). This proxy handles key generation, chat session encryption, and all communication with the IIP network. Point any standard IRC client at the isproxy, and it will connect you (anonymously and securely) to the IIP network.

IRC is a fairly simple system to use once a few basic commands are mastered. There are many IRC manuals and howtos on the Internet which will get you up to speed. Among the best are the documents located at IRCHelp.org, including the IRC Prelude, the IRC Primer and the step-by-step IRC Tutorial.

The Invisible IRC Project was created in order to allow anyone to chat with complete security and anonymity using a standard IRC client. IIP adds the following features to the tried-and-tested IRC architecture:

  • end-to-end and node-to-node encryption
  • fake traffic to thwart traffic analysis
  • three tiered network to protect the identities of the users and the server
  • working implementation of nickserv-like and chanserv-like service through "Trent" service
  • self-maintaining node reference files to ensure users are always able to connect to network
  • full traditional IRC support with the exception of CTCP and DCC: compatible with ALL IRC clients
  • OS support for all flavors of *nix including BSD, Linux, Mac OS X, and Solaris and Microsoft Windows 95 and up
  • node level flood control
  • GUI interface for Windows

In short, IIP takes a well tested and widely used Internet chat system and makes it private and secure.
